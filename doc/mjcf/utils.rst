:mod:`ikpy.mjcf.utils` module
----------------------------------------

Utility functions for MJCF parsing, including conversion between various
orientation representations (quaternion, axis-angle, Euler angles, etc.)
and RPY (roll-pitch-yaw) format used internally by IKPy.
